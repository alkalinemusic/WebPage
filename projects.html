Tally Counter
for product testing:


The inspiration for this app was taken from the tried and true method of counting an event's occurrence in fives; drawing 4 vertical lines and the 5th line being diagonal.
As a tester of tech equipment, the testing would often require repitions of the same method to determine when certain situations or failures would occur.
I saw my co-workers and myself sitting with pencil and paper placing 100's of these tally marks and knew that there could be a computerized solution.
This program is the result of this need.



MIDI Tool Set
for product testing:

This app is designed to take input from a MIDI device.
The MIDI data will then be scanned for any missing values in the 128 MIDI data set.
It is written in python and uses the python-rtmidi module.
