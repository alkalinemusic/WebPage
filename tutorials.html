Hello, Python!

October 22, 2018

Tradition has it that when first learning a new programming language,
displaying a message of 'Hello World!' is the first problem to be tackled.





For this tutorial, it is assumed that you have python installed.
If you're on OSX, it comes pre-installed.
For Windows users, you will have to install it on your own.
There are a multitude of resources for this on the web, and I have faith that you can do this!

First, you will need to open up IDLE.

Windows: press the Windows key - yes, that key between the Ctrl and Alt key.
Then start typing 'idle', and you should see the IDLE program show at the top of the list.
Highlight it and press enter.
OSX: press command plus the spacebar to open the spotlight search.
Once the search bar opens, type in 'idle', and you should see IDLE populate the search bar.
Press return

Once IDLE is open, you should see a window titled Python Shell.
If you do not see this window, select the Run>Python Shell and select that.
You should now see a heading in the window starting like this: Python 3.5.2 or similar numbers, followed by more info.

After this heading, you'll see the chevrons: >>>
This is where you will enter your code statements.

Yay!! finally on to the good stuff.
Now, type in the following exactly as it's written:

                     
    print('Hello, world!')

Then press enter.
The python shell should now display,
Hello, world!

And....that's it. You've just created your first python script!

What is happening here?
The word 'print' is telling the python shell that you want to display something to the screen.
Anything that you want to display to the screen needs to be enclosed within the parenthesis().

If we want to display text, which we've done here, you need to put that inside of quotes.
You can use single or double quotes, but you need to make sure you use one or the other for each print statement you write, not a mix of the two.

Practice: Now, take what you've learned and print out whatever statement you'd like, simply by using the line we created earlier.
All you need to do is change whatever is inside the quotes.

You can place almost any character you want inside those quotes.
Yes, I said almost any, because some characters are special.
But we'll leave that for another tutorial.
Happy printing!
